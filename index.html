Tamper-Resistant Smart Meter

  • Anti-theft enclosure & tamper sensors
  • NFC/ID card authentication for contractors
  • Real-time usage & balance data
  • Supports prepaid/postpaid, USSD recharge
Components/Tools:
  • Microcontroller (ESP32/STM32)
  • ATM90E36/ADE7753 metering ICs
  • NFC card reader, anti-tamper sensor
  • LoRaWAN/NB-IoT/PLC module
×

Renewable Smart Pole

  • Solar-powered energy relay hub
  • LoRaWAN/PLC/NB-IoT gateway
  • Environmental monitoring (temp, humidity, etc.)
  • Modular for lighting, security cameras, etc.
Components/Tools:
  • Solar panel & battery
  • Microcontroller (STM32/ESP32)
  • Sensor suite (temperature, motion, etc.)
  • LoRaWAN Gateway/Repeater
×

Passive Smart Node (PSN)

  • Clamps onto existing pole or cable
  • Solar and/or line-powered
  • Edge AI for tamper/theft detection
  • Low-cost, easy retrofit
Components/Tools:
  • MSP430/ESP32 microcontroller
  • Tamper/vibration sensor, solar cell
  • LoRaWAN/PLC module
  • QR code asset tracking
